# Laboratorio 7 Cifrado de Información

### Requisitos
Instalar las siguientes librerias

```shell
$ pip install elgamal
$ pip install pycryptodome
```

### Ejecución
Para la parte de Implementación sin librería ejecutar

```shell
$ python server.py
$ python client.py
```

Para parte Implementación con librería ejecutar 

```shell 
$ python parte2.py
```
